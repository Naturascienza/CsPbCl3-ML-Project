# CsPbCl3 Machine Learning Project

페로브스카이트 재료 CsPbCl3에 대한 머신러닝 분석 프로젝트입니다.

## 프로젝트 구조

```
CsPbCl3-ML-Project/
├── data/           # 데이터 파일
├── notebooks/      # Jupyter 노트북
├── src/            # 소스 코드
├── models/         # 학습된 모델
├── results/        # 결과 및 시각화
└── requirements.txt
```

## 설치

```bash
pip install -r requirements.txt
```

## 사용법

프로젝트 사용법은 추후 업데이트됩니다.
