### Known Challenges
1. **Incomplete reporting**: Some papers may not report all synthesis parameters
2. **Unit conversions**: Need to standardize units (mmol, ml, nm, etc.)
3. **Synthesis variations**: Multiple conditions in one paper (good for data!)
4. **Supplementary materials**: Critical data often in SI, not main text
5. **Terminology differences**: "Quantum dots" vs "nanocrystals" vs "perovskites"

### Solutions
- Use CrossRef API for metadata
- PDF text extraction for synthesis sections
- Manual review for complex cases
- Store raw data + normalized data separately
